How the data works. When this page is deployed with the price fetcher connected, the LIVE holdings pull real market prices and net worth = Σ(shares × real price) + private assets. With no connection (opened locally, or before setup), it falls back to a built-in simulator so it never looks broken — flip Demo mode to watch it move. Private holdings (Blue Origin, the Clippers, real estate) are labeled estimates from public records, not live. Share counts and private valuations still need final confirmation before publishing.
